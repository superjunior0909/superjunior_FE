# 0909 공구공구 프론트엔드

Vue 3 기반의 공동구매 플랫폼 프론트엔드 애플리케이션입니다.

## 프로젝트 개요

여러 판매자가 공동구매 상품을 등록하고, 소비자는 선착순/수량 조건에 맞춰 참여할 수 있는 마이크로서비스 기반 이커머스 서비스입니다.!

## 기술 스택

| 분류 | 기술 |
|------|------|
| **프레임워크** | Vue 3 (Composition API) |
| **라우팅** | Vue Router 4.3.0 |
| **HTTP 클라이언트** | Axios 1.12.2 |
| **빌드 도구** | Vue CLI 5.0 |
| **스타일링** | Scoped CSS |

## 주요 기능

### 사용자 기능
- **회원가입/로그인**: 이메일 인증 기반 회원가입
- **상품 탐색**: 카테고리별, 검색 키워드별 상품 탐색
- **공동구매 참여**: 진행 중인 공동구매 참여 및 현황 확인
- **장바구니**: 상품 추가, 수량 조정, 일괄 주문
- **주문**: 주소 관리, 포인트 충전 및 결제
- **프로필 관리**: 개인정보 수정, 주소록, 주문 내역
- **알림**: 실시간 알림 수신 및 읽음 처리
- **팔로우**: 판매자 팔로우 및 관리

### 판매자 기능
- **판매자 등록**: 판매자 신청 및 승인 프로세스
- **상품 관리**: 상품 등록, 수정, 삭제
- **공동구매 관리**: 공동구매 생성, 수정, 진행 상황 관리
- **판매 대시보드**: 판매 통계, 주문 관리
- **정산 관리**: 잔액 조회, 정산 내역 확인

### 커뮤니티 기능
- **게시글 작성**: 커뮤니티 게시글 작성 및 관리
- **사용자 디렉토리**: 다른 사용자 프로필 조회

## 시작하기

### 요구사항
- Node.js 14.x 이상
- npm 6.x 이상

### 설치

```bash
# 의존성 설치
npm install
```

### 개발 서버 실행

```bash
# 개발 서버 실행 (http://localhost:3000)
npm run serve
```

### 빌드

```bash
# 프로덕션 빌드
npm run build
```

### 린트

```bash
# ESLint 실행
npm run lint
```

## 환경 설정

### 개발 환경

`.env.development` 파일에서 백엔드 API 주소를 설정합니다:

```
VUE_APP_API_BASE_URL=http://localhost:8000
```

### 프로덕션 환경

프로덕션 빌드 시 백엔드 API 주소는 `https://0982.store`로 자동 설정됩니다.
